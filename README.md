## Installation

To get the tool clone the git repository:

```sh
git clnone git@github.com:Valeriisht/CodonOpimizatorTools.git
```
Create a conda environment with the necessary packages. 
Activate it:

```sh
conda env create -f environment.yml
conda activate 
```
